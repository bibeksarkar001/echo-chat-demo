# ECHO – Secure P2P Chat System

A lightweight, browser-based peer-to-peer communication platform designed for privacy and speed. ECHO leverages WebRTC to establish direct connections between users, ensuring that messages and media never touch a central server.

### 🚀 [Live Demo (Anonymous Version)]()

---

## 📖 Overview

**ECHO** is an end-to-end encrypted (E2EE) messaging and video calling prototype. By utilizing a decentralized architecture, it provides a secure environment for real-time communication without the need for traditional database storage for messages.

## ✨ Features

* **Anonymous Communication:** No sign-ups or personal data required. Simply choose a temporary username and start connecting.
* **Peer-to-Peer Messaging:** Send and receive text messages directly between browsers.
* **High-Quality Video Calling:** Built-in video and audio streaming via WebRTC.
* **Secure WebRTC Connection:** Encrypted data channels ensure privacy and security.
* **Contact Management:** Easily manage your list of active peers.
* **Browser Notifications:** Stay updated with incoming messages even when the tab isn't focused.
* **Modern Responsive UI:** A clean, intuitive interface built with mobile-first principles.

## 🛠️ Technologies Used

* **Frontend:** HTML5, JavaScript (ES6+)
* **Styling:** TailwindCSS
* **P2P Networking:** PeerJS
* **Communication Protocol:** WebRTC

## 🔒 Security & Privacy

Because ECHO is a **Peer-to-Peer (P2P)** system:

* **No Servers:** Your messages are not stored on any cloud database.
* **End-to-End Encryption:** The WebRTC standard inherently uses DTLS and SRTP to encrypt all data flowing between peers.
